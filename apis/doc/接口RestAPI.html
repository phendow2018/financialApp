/qichacha/orders post

post

建立一份订单
[uri - /api/v1/qichacha/orders]

post /qichacha/orders

建立一份订单
[uri - /api/v1/qichacha/orders]

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • companyName: required(string)

    企业名称

  • companyNumber: required(string)

    社会统一信用代码

  • type: required(one of 1, 2)

    订单类型
    [1 - 大数财务报表]
    [2 - 详细财务报表]

  • year: required(string)

    年份数组[用,隔开]

    Example:

    2018,2019
  • orderNumber: required(string)

    企查查订单号

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • code: required(integer)

    返回状态码
    [1 - 成功]
    [-1 - 失败]

  • msg: required(string)

    如果不成功，返回错误原因

  • orderNumber: required(string)

    成功下订单所生成的订单编号，后续访问可能需要用到
